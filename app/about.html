3D touch secure authentication -
using new mobile and desktop features to design Rubber Hose Secured authentication.

The following work has been done as part of the "Information Security Theory vs Reality" course given at Tel Aviv University. The application has been developed by Roy Michael and Eli Gerber BSc students at Tel Aviv University. Israel.
The application is used as a proof of concept for using new mobile and desktop features to design Rubber Hose Secured authentication.
