3D Touch Authentication

Welcome to 3D Touch Authentication training.
This application is a proof of concept for 3D touch authentication to be used as an implicit password sequence to preserve security against Rubber Hose Cryptanalysis. This should be used as a tool for a further study of the use of new mobile technologies as an implicit learning techniques.
The test score is calculated as a percent of how many points you hit at a certain threshold distance and also considers delta of the force level from the target

To start the training please choose training duration number of force points, distance and force threshold.
A default value would be assigned for empty input value.
